Bot discord

Ce mois d’avril, j’ai pu mettre en place un bot discord permettant de suivre ses statistiques sur le jeu Apex legends.

Codé en python, une bonne partie du programme consiste à mettre en forme des données, puis de les envoyer dans un chat. Mais j’ai me suis aussi appliqué sur la structure du code en utilisant la programmation objet. J’ai pu jouer avec des API et la librairie Mathplotlib pour créer des graphiques de progression.

J’ai pu ensuite mettre en ligne cette web app en l’hebergeant sur les services Microsoft Azure.

Même si le résultat n’est pas parfait, il est déjà bien au-dessus de mes attentes et me satisfait. Je continue encore à régler les derniers problèmes présents.
